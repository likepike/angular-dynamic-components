Пример динамической вставки компонента (ngx-dynamic-hooks)

Можно вставить компонент без параметра (должен вывести "EXAMPLE"):
<app-example></app-example>

Можно вставить компонент с передачей параметра text (должен вывести переданную строку)
<app-example [text]="'some text'"></app-example>

Можно даже вставить целиком корневой компонент приложения! И в нем тоже будет работать динамическая вставка!
<app-root></app-root>

Динамически создаваемые компоненты корректно создаются и уничтожаются (OnInit и OnDestroy выводятся в консоль).
Способ пригоден только для создания компонентов. Директивы, не создающие компонентов (в т.ч. структурные директивы NgIf, NgFor и т.п.) НЕ работают.

Динамически создаваемые компоненты должны быть явным образом указаны при инициализации парсера (см. app.module.ts). Неупомянутые там компоненты создаваться не будут.
